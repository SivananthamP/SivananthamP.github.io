# SivananthamP.github.io

Personal portfolio site built from the resume in this repository.

Preview locally:

```bash
python3 -m http.server 8000
# then open http://localhost:8000 in your browser
```

Deploy with GitHub Pages:

- This repository is named `SivananthamP.github.io`, so pushing the `main` branch will publish the site at https://SivananthamP.github.io
- Commit and push your changes, then wait a minute for Pages to become available.

Contact/links are included in `index.html`.
